App mobile iOS & Android

L'app pensée pour le terrain.

Les applications mobiles IO BILL sont des apps natives (pas une PWA), conçues pour les usages mobiles : créer une facture chez le client, scanner une facture fournisseur sur un chantier, faire signer un devis. Disponibles gratuitement pour tous les utilisateurs IO BILL.

⚡ La philosophie

L'app mobile n'est pas une version réduite du logiciel web. Elle expose les 5 actions qui comptent en mobilité : créer une facture, scanner un achat, faire signer un devis, voir sa trésorerie, répondre à un ticket. Le reste — paramétrages, déclarations, exports comptables — reste sur le web où c'est plus confortable.

1.Apps natives, pas une PWA

Beaucoup d'éditeurs SaaS proposent une PWA (Progressive Web App) sous le nom "app mobile" — c'est en fait juste leur site web encapsulé dans une icône. Cette approche a des limites réelles :

  • Performances dégradées (rendu navigateur dans une coquille) ;
  • Accès limité aux capteurs (caméra moins fluide, GPS approximatif) ;
  • Pas de notifications push fiables sur iOS ;
  • Pas de mode hors-ligne robuste.

IO BILL a fait le choix d'apps natives :

  • iOS : Swift / SwiftUI, support iPhone et iPad, exigence iOS 16+ ;
  • Android : Kotlin / Jetpack Compose, support smartphones et tablettes, exigence Android 11+ (API 30+).

Le résultat : un démarrage en moins de 1 seconde, un scanner de facture qui exploite les optimisations natives de l'appareil photo, et un mode hors-ligne réellement utilisable.

2.Créer une facture en 30 secondes

Le parcours de création d'une facture sur mobile est conçu pour être plus rapide qu'avec un papier :

  1. Tap sur le bouton "+" (toujours visible) ;
  2. Sélection du client (recherche par nom, par SIRET, ou création rapide avec auto-complétion SIRENE + VIES) ;
  3. Ajout de lignes de prestation (modèles enregistrés ré-utilisables en 1 tap) ;
  4. Validation de la TVA (auto-pré-remplie selon le profil client) ;
  5. Génération du PDF Factur-X en local, envoi via la PDP (ou par email pour B2C).

Pour les freelances qui facturent souvent les mêmes prestations, un profil de facture-type permet de générer une facture identique à la précédente en 2 taps (juste à modifier la date et la quantité).

3.Scanner une facture fournisseur

Sur le terrain, vous recevez une facture papier ou un ticket de caisse à intégrer dans la comptabilité ? L'app intègre un scanner optimisé :

  • Détection automatique des bords (le cadrage se fait tout seul) ;
  • Correction de perspective (la facture redevient un rectangle même si vous l'avez photographiée de biais) ;
  • OCR Mistral côté serveur : extraction des montants, TVA, fournisseur, date, lignes ;
  • Brouillon créé en 5 secondes, à valider d'un tap.

Plusieurs documents en série : la fonction "scan multi-pages" permet de photographier un ticket recto/verso, ou un lot de factures, en une seule passe. Chaque document devient une écriture distincte.

4.Faire signer un devis sur place

Vous êtes chez un client et vous venez de finaliser votre devis ? Pas besoin d'attendre le retour au bureau pour l'envoyer :

  1. Le devis est créé directement depuis le mobile ;
  2. Mode "signature en présence" : passez le téléphone au client, il signe avec le doigt sur l'écran ;
  3. Signature horodatée, position GPS optionnelle (pour les artisans BTP qui veulent prouver le lieu d'intervention) ;
  4. Valeur juridique eIDAS niveau "signature électronique avancée" via Yousign ;
  5. PDF signé envoyé immédiatement par email au client + facture pré-remplie en attente de votre validation.

Idéal pour les artisans, prestataires de service à domicile, commerciaux terrain.

5.Mode hors-ligne (chantier, route)

Pas de réseau ? Pas grave. L'app fonctionne en mode dégradé :

  • Création de devis et factures : les documents sont sauvegardés localement, synchronisés dès retour du réseau ;
  • Scan de factures fournisseurs : les photos sont prises et stockées localement ; l'OCR Mistral se déclenche au retour de la connexion ;
  • Consultation de la base clients : les 200 clients récents sont mis en cache pour les retrouver hors-ligne ;
  • Calcul de TVA : indépendant du réseau (logique intégrée dans l'app).

En revanche, les opérations qui requièrent obligatoirement le réseau (envoi via PDP, signature électronique distante, encaissement Stripe) sont mises en file d'attente et exécutées dès retour de la connexion.

6.Sécurité & biométrie

Vos données de facturation sont sensibles. L'app applique plusieurs couches de protection :

  • FaceID (iOS) / biométrie d'empreinte (Android) pour déverrouiller l'app à chaque ouverture, ou après inactivité ;
  • Code PIN local en repli (4 ou 6 chiffres) ;
  • Chiffrement local de la base hors-ligne (AES-256, clé dérivée du compte) ;
  • Verrouillage automatique après 2 minutes d'inactivité (paramétrable) ;
  • Effacement à distance en cas de perte du téléphone (depuis votre espace web) ;
  • Aucun export local non chiffré : les PDFs générés sont stockés dans le sandbox de l'app, pas dans la galerie photos.
