Facturation

La facturation française complète et conforme.

Du devis signé électroniquement à la déclaration TVA, en passant par l'OCR des achats et l'encaissement Stripe, IO BILL couvre l'intégralité du cycle de facturation d'une TPE française.

1.Documents émis

IO BILL gère tous les types de documents commerciaux et fiscaux courants en France :

  • Devis : modèle libre, mentions légales pré-remplies, durée de validité paramétrable, conversion en facture en un clic ;
  • Factures : Factur-X par défaut (PDF/A-3 + XML CII), toutes les mentions obligatoires de l'article 242 nonies A annexe II du CGI incluses, numérotation continue automatique selon votre exercice ;
  • Factures d'acompte : générées sur la base d'un devis ou d'une commande, intégration automatique dans la facture finale ;
  • Avoirs : avec ou sans remboursement, génération à partir d'une facture en deux clics, traçabilité complète ;
  • Situations de travaux (BTP) : facturation par étape selon avancement, intégration des retenues de garantie ;
  • Bons de commande et bons de livraison : pour les TPE qui ont besoin d'un cycle commande complet.

Tous les documents sont archivés automatiquement pendant la durée légale (10 ans pour les factures), avec coffre-fort numérique conforme aux exigences fiscales.

2.Signature électronique Yousign

Pour faire signer vos devis (ou tout autre document commercial) en ligne, en quelques clics, IO BILL intègre nativement Yousign :

  • Valeur juridique eIDAS (règlement européen 910/2014) — niveau "signature électronique avancée" pour la plupart des cas d'usage ;
  • Votre client reçoit un lien par email ou SMS, signe depuis son mobile en 30 secondes ;
  • Notification automatique côté IO BILL dès la signature ;
  • Le devis signé devient automatiquement une facture pré-remplie, prête à être validée.

Tarification : add-on à 5 € HT/mois pour les signatures illimitées, ou 1 €/signature à l'unité (pour les usages ponctuels).

3.Encaissement Stripe (Payment Link)

Sur chaque facture émise, IO BILL génère automatiquement un Payment Link Stripe sécurisé, intégré au PDF et accessible depuis le portail client :

  • CB (Visa, Mastercard, American Express, CB, Bancontact) ;
  • Virement SEPA instantané ou classique ;
  • SEPA Direct Debit (prélèvement) pour les abonnements récurrents ;
  • Apple Pay et Google Pay pour les transactions mobiles.

Le paiement reçu est automatiquement réconcilié avec la facture côté banque connectée. Vous recevez une notification immédiate.

Frais : tarification standard Stripe (1,4 % + 0,25 € par transaction CB en zone EU pour les commerçants français), aucune marge supplémentaire d'IO BILL.

4.OCR achats Mistral

Côté achats fournisseurs, IO BILL intègre un OCR développé sur l'API Mistral (modèle français hébergé en UE) qui extrait automatiquement :

  • Date, numéro de facture, identité du fournisseur (raison sociale, SIRET) ;
  • Lignes de détail, montants HT par ligne, taux et montants de TVA, total TTC ;
  • Conditions de paiement, échéance, IBAN.

Photographiez votre facture papier sur le terrain depuis l'app mobile, ou glissez-déposez un PDF depuis votre boîte mail : le brouillon de l'écriture est généré en 5 secondes. Vous validez (ou corrigez si nécessaire), c'est intégré.

OCR illimité à partir du plan Pro (limité à 20 documents/mois sur le plan Solo).

5.Boîte de réception PDP des achats

À partir de septembre 2026, vos fournisseurs B2B vous enverront leurs factures électroniques directement dans votre PDP (en utilisant votre SIRET). IO BILL centralise ces factures reçues dans une boîte de réception dédiée :

  • Réception en temps réel des factures électroniques structurées (Factur-X EN 16931 ou UBL) ;
  • Pré-traitement automatique : pas besoin d'OCR puisque les données sont déjà structurées ;
  • Validation, refus ou demande de modification en un clic ;
  • Intégration directe dans la comptabilité (achats) et dans le module TVA (TVA déductible).

Inclus dès le plan Solo.

6.GED des justificatifs (10 ans)

Tous vos documents (factures émises et reçues, justificatifs de notes de frais, bons de livraison, contrats) sont centralisés dans une GED native :

  • Rattachement automatique aux écritures correspondantes ;
  • Conservation 10 ans conformément à l'article L. 102 B du Livre des Procédures Fiscales ;
  • Recherche full-text dans le contenu des PDF (lecture OCR transparente) ;
  • Export en masse au format ZIP (pour archivage local ou contrôle fiscal).

7.Module TVA (CA3, CA12, franchise)

IO BILL gère les trois régimes TVA en vigueur en France :

  • Franchise en base de TVA (auto-entrepreneurs sous seuil) : suivi automatique du seuil, alerte de dépassement à 90 %, basculement assisté en cas de dépassement ;
  • Réel simplifié (CA12) : déclaration annuelle pré-remplie, acomptes semestriels suivis, ajustement de fin d'exercice ;
  • Réel normal (CA3) : déclaration mensuelle ou trimestrielle pré-remplie, contrôle de cohérence (TVA collectée vs TVA encaissée), export pour télédéclaration sur impots.gouv.fr.

Gestion multi-taux (20 %, 10 %, 5,5 %, 2,1 %, 0 %), autoliquidation intracommunautaire et hors UE, TVA sur encaissement ou sur débit selon votre choix d'option.

Inclus à partir du plan Pro (le plan Solo gère uniquement la franchise en base et le réel simplifié, suffisant pour la plupart des freelances).

8.Module URSSAF (micro, TNS, assimilé)

Selon votre régime social, IO BILL pré-calcule les cotisations dues :

  • Auto-entrepreneur (micro-entrepreneur) : cotisations sociales et impôt sur le revenu calculés selon votre type d'activité (BIC vente 12,3 %, BIC services 21,2 %, BNC 21,1 %, libéral CIPAV 21,2 % en 2026) — barèmes mis à jour automatiquement ;
  • TNS (Travailleur Non Salarié) au réel — entrepreneurs individuels, EIRL, SASU/SARL gérant majoritaire — cotisations URSSAF + RSI/SSI au réel, base de calcul automatisée ;
  • Assimilé salarié (gérant minoritaire SARL, président SAS) — fiches de paie déléguées à un outil dédié (Payfit, Silae) avec import des données de cotisations.

Échéances rappelées par notification (mensuelle, trimestrielle, semestrielle, annuelle). Export pré-rempli pour télédéclaration sur autoentrepreneur.urssaf.fr.

Module micro inclus dès le plan Solo. Modules TNS et assimilé inclus à partir du plan Pro.

9.API REST publique

Pour les utilisateurs avancés (TPE avec processus internes, intégrateurs, cabinets) qui souhaitent automatiser leur facturation :

  • API REST documentée selon la spécification OpenAPI 3.0 ;
  • Authentification par token Bearer avec rotation possible ;
  • Endpoints : factures, devis, clients, articles, paiements, banque, OCR, exports comptables ;
  • Webhooks sortants pour les événements clés (facture émise, facture payée, écriture lettrée, devis signé) ;
  • Rate limit : 60 requêtes/minute (plan Pro), 300 requêtes/minute (plan Cabinet).

Inclus à partir du plan Pro. Documentation publique sur https://api.iobill.online/docs (lien actif après lancement).

10.Intégrations Zapier & Make

Pour connecter IO BILL à votre stack sans écrire une ligne de code :

  • Zapier : 5 000+ apps disponibles (Notion, Slack, Google Sheets, HubSpot, Salesforce, Airtable, Trello, ClickUp, Asana...) ;
  • Make (ex-Integromat) : automatisations visuelles avancées avec routage conditionnel ;
  • n8n : intégration prévue Q3 2026 pour les utilisateurs auto-hébergés.

Cas d'usage typiques : "quand une facture est payée → ajouter une ligne dans Google Sheets et notifier Slack", "quand un nouveau client est créé → l'ajouter dans HubSpot avec son SIRET", "quand un devis est signé → créer une carte Trello pour le projet".
